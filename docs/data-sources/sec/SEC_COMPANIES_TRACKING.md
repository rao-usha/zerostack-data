## Query Ingested Data

To verify what data has been ingested, query the database:

```sql
-- See all SEC filings ingested
SELECT * FROM sec_10k ORDER BY filing_date DESC LIMIT 10;
SELECT * FROM sec_10q ORDER BY filing_date DESC LIMIT 10;
SELECT * FROM sec_8k ORDER BY filing_date DESC LIMIT 10;

-- Count filings by company
SELECT cik, company_name, COUNT(*) 
FROM sec_10k 
GROUP BY cik, company_name 
ORDER BY COUNT(*) DESC;

-- Check ingestion jobs
SELECT * FROM ingestion_jobs 
WHERE source = 'sec' 
ORDER BY created_at DESC;
```

## API Usage

To ingest a company's filings:

```bash
# Ingest Apple's 10-K and 10-Q filings from 2020-2024
curl -X POST "http://localhost:8001/api/v1/sec/ingest/company" \
  -H "Content-Type: application/json" \
  -d '{
    "cik": "0000320193",
    "filing_types": ["10-K", "10-Q"],
    "start_date": "2020-01-01",
    "end_date": "2024-12-31"
  }'

# Ingest multiple companies at once
curl -X POST "http://localhost:8001/api/v1/sec/ingest/multiple" \
  -H "Content-Type: application/json" \
  -d '{
    "ciks": ["0000320193", "0000789019", "0001652044"],
    "filing_types": ["10-K", "10-Q"],
    "start_date": "2020-01-01",
    "end_date": "2024-12-31"
  }'
```

## Notes

- **Default Date Range:** Last 5 years if not specified
- **Default Filing Types:** 10-K and 10-Q if not specified
- **Rate Limits:** SEC enforces 10 requests/second; this service uses 8 req/sec to be conservative
- **Data Source:** All data is from SEC EDGAR (https://www.sec.gov/edgar)
- **License:** SEC data is public domain
- **Job Tracking:** All ingestion runs are tracked in `ingestion_jobs` table
